# srcwalk

Code navigation for AI agents: exact reads, structural discovery, callers/callees, dependency evidence, and bounded context/review packets.

```sh
npm install -g srcwalk
srcwalk guide
```

The npm installer downloads the matching binary from the same-version GitHub release, requires HTTPS throughout redirects, verifies the published SHA-256 checksum, validates the single-file archive shape, and only then installs the binary.

For usage, supported languages, examples, source installation, and release notes, see the [srcwalk repository](https://github.com/sting8k/srcwalk).
